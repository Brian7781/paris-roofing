Paris Roofing Services — Images Folder
======================================

Drop real project photos into this folder. Recommended:

1. og-cover.jpg       1200x630  — social sharing image (referenced in Open
                                    Graph + Schema.org)
2. hero.jpg           1920x1080 — optional hero background photo (wire into
                                    .hero in css/style.css via background-image)
3. roof-before-1.jpg  / roof-after-1.jpg — before/after project pairs
4. crew.jpg           — a crew shot builds real trust
5. truck.jpg          — company truck with phone number visible

File naming tips:
- All lowercase, hyphens-not-spaces
- Keep files under 300KB (use https://squoosh.app or `cwebp`)
- Prefer .jpg for photos, .svg for icons

When you drop photos in, reference them in index.html like:
  <img src="images/roof-before-1.jpg" alt="Hail-damaged roof in Georgetown, TX before replacement" />

Always write a descriptive alt attribute — it helps SEO and accessibility.
